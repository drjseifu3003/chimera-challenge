# Project Chimera - Specification Meta

## Overview
This directory contains the meta-specification and planning artifacts for Project Chimera, following the Git Spec Kit structure. It provides traceability, rationale, and planning context for all functional and technical specifications.

## Structure
- functional.md: User stories, requirements, and acceptance criteria
- data-model.md: Entity and relationship definitions
- contracts/: API and integration contracts (OpenAPI, GraphQL, etc.)
- quickstart.md: Setup and onboarding guide
- _meta/: Meta-specification, planning, and constitution

## Traceability
Each specification artifact is versioned and linked to its corresponding feature, rationale, and research. All changes are tracked via Git for full auditability.

## Constitution
The constitution defines the principles, constraints, and non-negotiable rules for the project. See constitution.md for details.
